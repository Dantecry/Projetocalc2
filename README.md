# Sobre

Calculadora que se conecta a um Servidor que executa as operações
através do uso de RMI:

Servidor 1 : 4 Operações básicas.

Soma: num1 + num2

Subtração: num1 - num2

Multiplicação: num1 * num2

Divisão: num1 / num2

Servidor 2 : 3 operações complexas.

Porcentagem

Raíz quadrada

Potenciação
